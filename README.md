# WP Easy Email
A wordpress plugin for smtp/imap/pop3 client.

# Feautures
-> SMPTP/ IMAP / POP3 support 

-> Inline attachments, attachments support

-> Form builder

-> Email tempalate creator

-> Two IMAP/POP3 engines for better performance on different platforms
